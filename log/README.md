## 🔍 로그 확인 방법

### 최근 QA 결과 보기
```bash
tail -50 log/$(date +%Y-%m-%d).md
```

### 오늘 헬스 체크 기록
```bash
grep "헬스 체크" log/$(date +%Y-%m-%d).md
```

### 자동 개선 제안 보기
```bash
tail -100 log/improvement-log.md
```

### 정확도 트렌드 분석
```bash
grep "정확도" log/2026-02-*.md
```
